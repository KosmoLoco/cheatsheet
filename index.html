HTML Cheatsheet

HTML Tags

Tag Name Description
<html> HTML Tag that is used to specify that the document is HTML.
<head> Head Used to contain all head elements of the file, which include title, style, meta, etc...
<title> Title Defines the title of the document. Displays in browser tabs.
<a href="..."> Anchor Used to link one page to another page
<ul>, <ol> Lists UL used to create list without order. OL used to create an ordered list.
<br> Break Used to break the line.

CSS Selectors

Selector Name Description
. Class (.) Selects elements with a unique class attribute. For example .intro styles all elemtns with class="intro"
* Universal (*) Selects all elemtns on the page and styles them.
p, h1, a Elements Selects and styles all specified elements, such as <p>, <h1>, <a>
# ID (#) Used to select a unique targeted element.
[attribute="value"] Attribute Selects elements of a page based on their value. For example, [title="my title"] would find the coresponding tag in the html file.
